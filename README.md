# ExpenseManager
 Kotlin Mobil Application

Költség és bevásárlás követő mobil alkalmazás Kotlin nyelven, lehet rajta bevásárlólistát követni és a költségeket számontartani, amihez grafikonok is segítséget nyújtanak.
